Audio Guidelines

  • Audio must be clear and free of unrelated background noise/speech. Extremely noisy recordings may be rejected.
  • If the sound contains speech, the description must contain the full transcription in its original language.
    • For sound effects, unless there is a name for it (e.g. Montréal's "Dou dou dou", Japan's departure melody song names), leave the description blank.
    • If a name does exist, please wrap it in the native language's quotation marks.
  • Trim guidelines:
    • Trim must start at the exact beginning of said sound. There must not be any background noise before the sound plays.
    • Trim must end at any reasonable time after the end of said sound, usually around 1-2 seconds. Crossfade is optional but recommended.
  • Try to name the file something that relates to the sound it represents.

Notes

  • Submissions are reviewed before they are added to the atlas.
  • If you would like to see a country or system added, please open a GitHub issue.
  • Sounds uploaded must be currently used/most recently known used sound on the system.
  • Before submitting, please check whether the sound already exists in the atlas to avoid duplicates.
